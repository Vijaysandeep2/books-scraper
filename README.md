# 📚 Books Scraper — Python Web Scraping with Scrapy & MongoDB

A web scraper built with Python and Scrapy that extracts book data and stores it in MongoDB.

## 🚀 Tech Stack
- Python 3.10+
- Scrapy
- MongoDB
- PyMongo

## 📦 Features
- Scrapes title, price, rating, availability of 1000+ books
- Handles pagination automatically
- Stores data in MongoDB
- Exports to JSON and CSV
